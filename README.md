# mainline – Mainline is a free template built with shadcn/ui, Tailwind 4 and Next.js 15.

#### Preview

 - [Demo](https://themewagon.github.io/mainline/)

#### Download
 - [Download from ThemeWagon](https://themewagon.com/themes/mainline/)

## Getting Started

1. Clone Repository
```
git clone https://github.com/themewagon/mainline.git
```
2. Install Dependencies
```
npm i
```
3. Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
